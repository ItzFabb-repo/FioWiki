# Leaderboards

This page explains a simple setup flow for Sawit's leaderboard system so server owners can avoid the kind of error shown in `Leaderboard Step 1`.

Current verified plugin references connected to leaderboard-style features:

| Plugin | Version | Role |
| --- | --- | --- |
| `ajLeaderboards` | `2.11.0` | Ranking displays and stats boards. |

## Step 1

This image below if your leaderboard is not setted up-correctly.

![Leaderboard Step 1](/img/setup/sawit/leaderboard-step-1.png)

## Step 2

Use the following command `/ajlb add [TAB]`

`[TAB]` means that you must press the `TAB` button on your keyboard.

![Leaderboard Step 2](/img/setup/sawit/leaderboard-step-2.png)

## Step 3

Use this as the final target state for the leaderboard setup.

![Leaderboard Step 3](/img/setup/sawit/leaderboard-step-3.png)

:::tip[Recommendation]
If the leaderboard shows errors like in Step 1, follow the setup flow from Step 1 to Step 3 in order instead of skipping directly to the final state.
:::

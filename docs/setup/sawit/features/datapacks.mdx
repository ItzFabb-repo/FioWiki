# Datapacks

![Sawit Datapacks](/img/setup/sawit/sawit-datapacks.png)

This page lists the datapacks detected in the current Sawit package.

## Detected Datapacks

| Datapack | Version | Type | Description | Direct link |
| --- | --- | --- | --- | --- |
| `Tectonic` | `3.0.22` | World generation | Large-scale terrain generation datapack used to reshape overworld terrain. | [Download / Project](https://modrinth.com/datapack/tectonic) |
| `Terralith` | `2.6.2` | World generation | Biome and terrain expansion datapack for more varied overworld generation. | [Download / Project](https://modrinth.com/datapack/terralith) |
| `Vanilla Refresh` | `1.4.30` | Vanilla enhancement | Vanilla-style datapack enhancement package included in the Sawit resource set. | [Download / Project](https://modrinth.com/datapack/vanilla-refresh) |
| `bukkit` | bundled | Plugin support | Internal Bukkit-generated datapack for resources provided by plugins. | Built into the server package |

## Notes

| Item | Detail |
| --- | --- |
| Datapack folder | `RESOURCE/datapacks` |
| Bukkit pack | The `bukkit` datapack is not a third-party gameplay pack. Its `pack.mcmeta` describes it as `Data pack for resources provided by Bukkit plugins`. |
| Worldgen focus | The main third-party gameplay/worldgen datapacks currently detected are `Tectonic`, `Terralith`, and `Vanilla Refresh`. |

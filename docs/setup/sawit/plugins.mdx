## Quick Assessment

This plugin stack is broad and feature-rich. It is built more like a content-heavy survival network than a minimal SMP.

:::info[Layout]
The server also bundles `spark` for profiling, but it is treated as a helper component rather than a separate plugin entry.
:::

## Watch List

These are the plugins I would keep under closer observation:

- `Essentials`
  Running a dev build instead of a stable release tag.

- `QuickShop-Hikari`
  Snapshot build, so compatibility should be checked after updates.

- `LagFixer`
  Performance helper plugins should be tested carefully because aggressive cleanup behavior can affect stability or gameplay.

- `packetevents`
  Shared packet library used by multiple plugins, so it is worth watching after version bumps.

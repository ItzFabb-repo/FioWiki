# Inventory See Config

Source file:

```text

```

## What this module covers
- `/invsee <player>` inventory inspection,
- `/ecsee <player>` ender chest inspection,
- staff-side inspection flow for moderation or support.

  - `fiochat.inventorymanagement.invsee.edit`

# How to Create

Create one file per voucher inside `plugins/FioVoucher/vouchers/`.

Basic flow:

1. copy one of the sample files
2. rename it to your voucher id
3. set `voucher.id`
4. set the item
5. define the reward commands
6. set redeem rules
7. reload the plugin

Minimal shape:

```yaml
voucher:
  id: example
  display-name: "Example Voucher"
  enabled: true
  item:
    material: PAPER
    amount: 1
    name: "Example Voucher"
    lore:
      - "Voucher"
  commands:
    claimed:
      enabled: true
      actions:
        - "[MESSAGE] Reward claimed"
  settings:
    trial:
      enabled: false
      duration: "0s"
      allow-pause: false
      allow-stack: false
    redeem:
      max-claim: 1
```

For source-backed examples, see [Example](/plugin/fiovoucher/vouchers/example).

## External References

Use these references when choosing valid values:

- Voucher item materials: [Spigot API: `org.bukkit.Material`](https://hub.spigotmc.org/javadocs/spigot/org/bukkit/Material.html)
- Item tooltip flags: [Spigot API: `org.bukkit.inventory.ItemFlag`](https://hub.spigotmc.org/javadocs/spigot/org/bukkit/inventory/ItemFlag.html)
- Sounds for `[SOUND]`: [Spigot API: `org.bukkit.Sound`](https://hub.spigotmc.org/javadocs/spigot/org/bukkit/Sound.html)
- Enchantments for `[ITEM]` enchantment tokens: [Spigot API: `org.bukkit.enchantments.Enchantment`](https://hub.spigotmc.org/javadocs/spigot/org/bukkit/enchantments/Enchantment.html)
- MiniMessage text formatting: [PaperMC Docs: MiniMessage format and standard tags](https://docs.papermc.io/adventure/minimessage/format/)
- MiniMessage overview: [PaperMC Docs: MiniMessage](https://docs.papermc.io/adventure/minimessage/)

Prefer the official Spigot/Paper documentation over copied lists from forums, because enum names and supported tags can change between Minecraft versions.

# Flags

`item.flags` controls which item tooltip flags are hidden.

Example:

```yaml
item:
  flags:
    - HIDE_ATTRIBUTES
    - HIDE_ENCHANTMENTS
    - HIDE_DYED_COLOR
    - HIDE_DESTROYS
```

Source examples also use:

- `HIDE_UNBREAKABLE`

Use flags when you want the voucher item to look clean in inventory.

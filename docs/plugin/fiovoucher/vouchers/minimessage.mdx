# MiniMessage

Voucher text uses MiniMessage formatting.

Common tags in the source:

- `<gray>`
- `<gold>`
- `<yellow>`
- `<dark_gray>`
- `<green>`
- `<red>`
- `<white>`
- `<italic>`
- `<bold>`
- `<gradient:...>`
- `<shadow:...>`

You can also use line breaks inside action strings and lore text.

Example:

```yaml
name: "<gradient:#95D5B2:#40916C>Random Food Voucher</gradient>"
```

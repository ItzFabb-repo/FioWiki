# Gradients

The source uses gradients in both `display-name` and `item.name`.

Example:

```yaml
display-name: "<gradient:#FFD166:#F77F00>Money Voucher</gradient>"
```

Gradient names in the bundle usually combine:

- a bright start color
- a darker end color
- an optional shadow tag

This is the main style used by the sample voucher files.

## External reference

Gradients are standard MiniMessage tags. Use the official MiniMessage format page for syntax, phase arguments, and related tags:

- [PaperMC Docs: MiniMessage format and standard tags](https://docs.papermc.io/adventure/minimessage/format/)

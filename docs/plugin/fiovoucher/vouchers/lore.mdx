# Lore

`item.lore` is the text shown under the voucher name.

Typical structure:

```yaml
lore:
  - "<dark_gray>Voucher"
  - ""
  - "<gray>Short description"
  - ""
  - "<yellow>Click to redeem this voucher"
```

The source examples use lore to show:

- what the voucher does
- what reward it gives
- how to claim it
- whether it is a trial or a limited-use voucher

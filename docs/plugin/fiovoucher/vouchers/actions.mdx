# Actions

Voucher file actions are bracketed commands executed after a claim result.

Supported action ids in the source:

- `[MESSAGE]`
- `[SOUND]`
- `[CONSOLE]`
- `[ITEM]`
- `[EXP_LEVELS]`
- `[EXP_POINTS]`
- `[ACTIONBAR]`
- `[TITLE]`
- `[SUBTITLE]`
- `[TOAST:<material>]`

Example:

```yaml
actions:
  - "[MESSAGE] <gray>You got the reward.</gray>"
  - "[CONSOLE] eco give %player_name% 1000"
  - "[ITEM] DIAMOND 3"
```

## External references

Some action arguments are standard Bukkit/Spigot enum names or MiniMessage text:

- Materials in `[ITEM]` and `[TOAST:<material>]`: [Spigot API: `org.bukkit.Material`](https://hub.spigotmc.org/javadocs/spigot/org/bukkit/Material.html)
- Sounds in `[SOUND]`: [Spigot API: `org.bukkit.Sound`](https://hub.spigotmc.org/javadocs/spigot/org/bukkit/Sound.html)
- MiniMessage text in `[MESSAGE]`, `[ACTIONBAR]`, `[TITLE]`, `[SUBTITLE]`, and `[TOAST]`: [PaperMC Docs: MiniMessage format and standard tags](https://docs.papermc.io/adventure/minimessage/format/)

# Race Explanation

## Shared race structure

Most race files follow this shape:

```yaml
game: RACE
enable: true
points: 1
time_seconds: 300
race:
  type: BREAK
  material: STONE
  amount: 10
  top-reward-count: 3
  top-announce-count: 3
  bossbar:
    enabled: true
```

Many race files then add objective-specific sections such as:

- `entity`
- `enchant`
- `placeholder`
- `shoot-at-entity`
- `smelt`
- `trading`
- `obtain-items`
